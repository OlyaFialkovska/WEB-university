**Тема:** Розробка функціональності Web-додатка мовою Javascript

## Завдання
Розробити функціональність для статичних сторінок Web-додатка першої лабораторної роботи із використанням шаблону MVC.

## Назва додатка
MindVault — Web-додаток для збереження ідей, цілей, планів та мотиваційних нотаток.

## Реалізовано
- сторінка реєстрації користувача;
- сторінка входу;
- сторінка профілю користувача;
- сторінка «Про додаток»;
- робоча сторінка з архітектурою MVC;
- збереження даних через `localStorage`;
- додавання, редагування, видалення, пошук і фільтрація записів.

## Структура Javascript
- `models/` — моделі даних;
- `views/` — відображення робочої сторінки;
- `controllers/` — керування подіями та взаємодією між model і view.
