# Лабораторна робота №3

**Студентка:** Ольга Фіалковська  
**Група:** КВ-34  
**Дисципліна:** Web-дизайн  
**Тема:** Розробка Web-додатка засобами VueJS з асинхронною взаємодією клієнта і сервера

## Мета роботи
Ознайомитися із засобами фреймворка VueJS, навчитися створювати асинхронні запити до Web-сервера та організувати збереження даних у базі даних SQLite.

## Що виконано
- програмний код ЛР №2 адаптовано під VueJS;
- створено сервер на NodeJS та ExpressJS;
- реалізовано збереження користувачів і записів у SQLite;
- реалізовано асинхронну взаємодію клієнта і сервера через fetch API;
- реалізовано реєстрацію, вхід, профіль користувача та робочу сторінку з CRUD-операціями для нотаток.

## Структура проєкту
- `server.js` — сервер Express;
- `database/db.js` — підключення та ініціалізація SQLite;
- `public/` — клієнтська частина на VueJS;
- `public/js/` — модулі для API, авторизації та сторінок;
- `public/css/style.css` — стилі додатка.

## Використані технології
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- VueJS 3
- NodeJS
- ExpressJS
- SQLite
- fetch API

## Інструкція запуску
1. Відкрити термінал у папці проєкту.
2. Виконати команду `npm install`.
3. Виконати команду `npm start`.
4. У браузері перейти за адресою `http://localhost:3000`.

## Основні маршрути API
- `POST /api/register` — реєстрація користувача;
- `POST /api/login` — вхід у систему;
- `GET /api/users/:id` — отримання профілю користувача;
- `GET /api/users/:id/stats` — отримання статистики;
- `GET /api/users/:id/notes` — отримання списку записів;
- `POST /api/users/:id/notes` — створення запису;
- `PUT /api/users/:id/notes/:noteId` — редагування запису;
- `DELETE /api/users/:id/notes/:noteId` — видалення запису.

## Висновок
У ході виконання лабораторної роботи було розроблено повноцінний клієнт-серверний Web-додаток. Використання VueJS спростило керування станом інтерфейсу, а ExpressJS та SQLite забезпечили серверну логіку та надійне збереження даних.
